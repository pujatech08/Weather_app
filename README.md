# 🌦️ Weather App

A simple and responsive Weather App that provides real-time weather information for any city using the OpenWeather API.

---

## 🚀 Features

* 🌍 Search weather by city name
* 🌡️ Displays temperature, humidity, and weather conditions
* ⛅ Dynamic weather updates
* 📱 Responsive and user-friendly UI

---

## 🛠️ Tech Stack

* HTML
* CSS
* JavaScript
* OpenWeather API

---

## 🔗 API Used

This project uses the **OpenWeather API** to fetch real-time weather data:
👉 https://openweathermap.org/

---

```
Weather_app/

```

---

## ⚙️ How to Run Locally

1. Clone the repository:

   ```bash
   git clone https://github.com/pujatech08/Weather_app.git
   ```

2. Open the project folder:

   ```bash
   cd Weather_app
   ```

3. Open `index.html` in your browser

---

## 🔑 Setup API Key

1. Go to https://openweathermap.org/ and create an account
2. Generate your API key
3. Replace the API key in your `script.js` file:

   ```javascript
   const apiKey = "YOUR_API_KEY";
   ```
